53 

54 This is used when reading a file or fetching an URL. 

55 The character encoding is determined from the initial bytes 

56 (a :abbr:`BOM (Byte Order Mark)` or an ``@charset`` rule) 

57 as well as the parameters. 

58 The ultimate fallback is UTF-8. 

59 

60 :param css_bytes: A byte string. 

61 :param protocol_encoding: 

62 A string. 

63 The encoding label, if any, defined by HTTP or equivalent protocol. 

64 (e.g. via the ``charset`` parameter of the ``Content-Type`` header.)

67 for the `environment encoding`_, 

68 if any. 

69 :param skip_comments:

71 If the input is a string, ignore all comments. 

72 :param skip_whitespace: 

73 Ignore whitespace at the top-level of the stylesheet. 

74 Whitespace is still preserved 

75 in the :attr:`~tinycss2.ast.QualifiedRule.prelude` 

76 and the :attr:`~tinycss2.ast.QualifiedRule.content` of rules. 

77 :returns: 

78 A ``(rules, encoding)`` tuple. 

79 

80 * :obj:`rules` is a list of 

81 :class:`~tinycss2.ast.QualifiedRule`, 

82 :class:`~tinycss2.ast.AtRule`, 

83 :class:`~tinycss2.ast.Comment` (if ``skip_comments`` is false), 

84 :class:`~tinycss2.ast.WhitespaceToken` 

85 (if ``skip_whitespace`` is false), 

86 and :class:`~tinycss2.ast.ParseError` objects. 

87 * :obj:`encoding` is the :class:`webencodings.Encoding` object 

88 that was used. 

89 If ``rules`` contains an ``@import`` rule, this is 

90 the `environment encoding`_ for the imported stylesheet. 

91 

92 .. _environment encoding: 

93 http://www.w3.org/TR/css-syntax/#environment-encoding 

94

96

98 rules, encoding = parse_stylesheet_bytes( 

99 css_bytes=response.read(), 

100 # Python 3.x 

101 protocol_encoding=response.info().get_content_type().get_param('charset'), 

102 # Python 2.x 

103 protocol_encoding=response.info().gettype().getparam('charset'), 

104 ) 

105 for rule in rules: 

106 ... 

107 

108 """ 

109 css_unicode, encoding = decode_stylesheet_bytes(

111 stylesheet = parse_stylesheet(css_unicode, skip_comments, skip_whitespace)
